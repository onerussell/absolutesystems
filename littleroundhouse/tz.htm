Пространство страницы условно разделено на 5 столбцов (крайний правое и крайний левый должны совпадать по ширине) (см рис. 1)

 

Краткое описание разделов:

 

1. Реклама (текстовые блоки что-то типа google adwords)

2. Groups – группы. Список групп  которые могут формировать зарегистрированные участники форума. Выводим несколько для примера и внизу  линк “more”. Название каждой группы представлено виде линка который ведет к странице Group Summary.

3. Action . Содержит в себе несколько категорий (Pools,Petitions,Letters/Calls,Resumes,Projects,Elections) смысл которых расшифрован ниже в соответствующем разделе.

4. Events. Последние несколько записей из таблицы Events и линк на полный список событий.

5. My Profile ( поля для ввода логина, пароля, восстановление забытого пароля и регистрация новых пользователей) Внизу оставшееся пространство предполагается забить текстовыми объявлениями  рекламного характера …такого же плана как на самой левой полосе.

Человек создавший группу автоматически становиться модератором группы и может наделять правами модератора других членов группы.

Создатель (Администратор группы) единственный кто может

·        расформировать группу

·        назначать другого члена группы модератором

 

 

Существует 4 различные ситуации

1.      Отсутствие модерирования как такового…постинги разрешаются абсолютно всем (для этого даже не нужно являться членом группы)

2.      Постинги членов группы публикуются напрямую без вмешательства модератора

3.      Только постинги членов группы помеченных как “trusted” публикуются напрямую…остальные должны быть утверждены модератором

4.      Абсолютно все постинги без исключения модерируются прежде чем быть опубликованными на сайте.

 

      Владелец группы в любой момент может перейти от одной схемы модерирования в другой.

 

1.2  Присоединение к существующей группе

 

5.      Выбираем группу из списка – попадаем на страницу с кратким описанием группы

 После присоединения к группе пользователь получает возможность постить сообщения. Здесь функционал во многом напоминает phpbb

и аналогичные форумы. (edit, reply, quote, signature)

 

1.3  Создание подгрупп (subgroup)

 

Администратор одной группы может обратиться к администратору другой группы на предмет создания сообщества в котором одна из групп будет выступать в роли подгруппы. Для решение этого вороса необходимо обоюдное решение сторон. То есть  Администратор одной из групп должен пометить другую группу как дочернюю в свою очередь Администратор другой группы помечает другую как родительскую.

 

My Contacts

 

Содержит информацию собранную в процессе регистрации…которую соответственно можно изменить с этой страницы. Помимо полей которые были введены на этапе регистрации добавляется еще signature – автоматическая подпись которая добавляется к каждому постингу данного пользователя.

> Actions menu:
>
> Polls:  This is a place to list online polls.  Any member can log on, and 
> list a poll.  It consists of three fields: a title, a description (500 
> words) and a url for the poll.  This can be a group poll or an outside 
> media (ie Newsweek or nyt) poll.
>
> Petitions: This is a place to list online petitions.  Any member can log 
> on, and list a petition.  It consists of three fields: a title, a 
> description (500 words) and a url for the petition.  Again, the url can 
> point anywhere.
>
> Letters/Calls: This is a place to list writing/calling campaigns.  Any 
> member can log on, and list a campaign.  It consists of three fields: a 
> title, a description (500 words) and a phone number or mailing address.
>

> Elections:  This is a place to list elections.  Any member can log on, and 
> list an election.  It consists of three fields: a title, a description 
> (500 words) and a url, phone number or mailing address of the candidate or 
> the group that is organizing an election effort.  Also, members can assign 
> field values to their election to show the reach of the official being 
> elected.
